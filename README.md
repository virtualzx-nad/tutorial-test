# MNIST Tutorial

This tutorial provides an introduction to building and training neural nets with [keras](http://keras.io).

The "Hello World!" of machine learning is the MNIST data set of images of hand drawn digits. 
Here we practise building a DNN with 1 hidden layer (so not very deep) and a (very vanilla) CNN.


## Instructions

Download Anaconda for Python 2.7 [here](https://repo.anaconda.com/archive/Anaconda2-2018.12-MacOSX-x86_64.pkg)

Clone repo
```
	$ git clone git@bitbucket.org:mtzcloud/ml-tutorials.git 
	$ cd ml-tutorials/MNIST_tutorial
```
Create and activate environment
```
	$ conda env create -f ML-tutorial-MNIST.yml
	$ conda activate ML-tutorial-MNIST 
```
Launch Notebook
```
	$ jupyter notebook 
```
Run the code 

Visualise with Tensorboard
```
	$ tensorboard –-logdir=logs
```

Tutorial information in Slides_Tutorial_MNIST.pptx
